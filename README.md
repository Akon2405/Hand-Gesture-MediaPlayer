# Hand Gesture Media Player

A real-time computer vision script that controls computer media playback and volume using hand gestures. Built with Python, OpenCV, and the MediaPipe Tasks API.

## Setup Instructions
1. Clone this repository.
2. Install dependencies: `pip install -r requirements.txt`
3. **Important:** Download the official [MediaPipe Hand Landmarker Model (float16)](https://storage.googleapis.com/mediapipe-models/hand_landmarker/hand_landmarker/float16/1/hand_landmarker.task) and place the `hand_landmarker.task` file in the root directory.
4. Run `python mediaplayer.py`.
